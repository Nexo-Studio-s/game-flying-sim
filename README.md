# Nexo Studio's Flying Sim

**Nexo Studios – Prototype Roblox flying simulator project**  
Early development and experimentation.

## Overview
This project is a Roblox flying simulator designed to experiment with game mechanics and gameplay systems.  
The repository contains all scripts, assets, and Roblox Studio files necessary for development.

## Getting Started
1. Clone or download this repository.
2. Open Roblox Studio.
3. Load the project from the downloaded files.
4. Explore the scripts and assets to understand the game mechanics.
